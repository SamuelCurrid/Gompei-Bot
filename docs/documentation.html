Commands

Default prefix is "."
Command Description Permissions Required
help Standard help message None
messages Sends the messages leaderboard None
quotes Sends the quotes leaderboard None
reactions Sends the reactions leaderboard None
reset Resets all leaderboard values Administrator
set CHANNEL_MENTION Sets the quotes channel for the guild Administrator
prefix NEW_PREFIX Sets the command prefix Administrator
logging CHANNEL_MENTION Sets the logging channel for the guild Administrator
