Commands

Default prefix is "."
Command Description Permissions Required
help Standard help message None
messages Sends the messages leaderboard None
quotes Sends the quotes leaderboard None
reactions Sends the reactions leaderboard None
emojis Sends the emojis leaderboard None
echo CHANNEL_MENTION MESSAGE Makes Gompei echo message to given channel Administrator
reset Resets all leaderboard values Administrator
set CHANNEL_MENTION Sets the quotes channel for the guild Administrator
prefix NEW_PREFIX Sets the command prefix Administrator

Movie Poll Commands

Command Description Permissions Required
addMovie MOVIE_NAME Adds the movie of certain name to the poll None
movieInfo MOVIE_NAME Sends the info for a movie None
vote MOVIE_NAME Vote for a movie None
removeVote MOVIE_NAME Remove your vote for a movie None
listMovies Lists the current movies in the poll None
removeMovie MOVIE_NAME Removes a movie from the poll Administrator
